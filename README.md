# intellij-sbt-tasks

Intellij Idea run configurations for Scala SBT projects:
- compile (Test/compile)
- recompile (clean Test/compile)
- publishLocal
- tests
- integrationTests
- ...

### Installation

```scala
addSbtPlugin("pl.newicom.ide.intellij" % "intellij-sbt-tasks" % "<current version>")
```
